# C++算法答题打怪游戏 - 设计文档

**创建日期**: 2026-04-16
**版本**: 1.0
**项目类型**: 在线多人算法竞赛游戏

## 项目概述

这是一个在线C++算法竞赛游戏，玩家通过回答算法题来击败怪物，提升段位，与其他玩家进行实时竞技。游戏将算法学习与RPG元素结合，提供寓教于乐的学习体验。

## 核心功能

### 1. 游戏模式

#### 单人模式
- 玩家按段位系统逐步挑战怪物
- 每答对一题，怪物扣血
- 答错则怪物回复部分血量
- 击败怪物后提升段位

#### 多人模式
- 创建房间或加入公开房间
- 支持2-10人同时游戏
- 两种子模式：
  - 个人竞技：独立答题，比较完成速度和准确率
  - 团队模式：2v2，团队总血量共享

### 2. 段位系统

**段位层次（从低到高）：**
1. D → D+
2. C → C+
3. B → B+
4. A → A+
5. 青铜S
6. 黄金S
7. 钻石S
8. 魔王S

**段位提升规则：**
- 胜利2场：提升1星（D→D+）
- D+获胜：晋升C
- S级获胜：提升1星（青铜S→黄金S→钻石S→魔王S）
- 连胜3次：额外获得1星
- 失败3场：降低1星或段位

### 3. 怪物系统

**段位对应怪物：**
- D/D+：史莱姆（HP 30）
- C/C+：哥布林（HP 50）
- B/B+：兽人（HP 80）
- A/A+：暗影战士（HP 120）
- 青铜S：熔岩巨人（HP 150）
- 黄金S：雷霆龙（HP 180）
- 钻石S：神圣龙（HP 220）
- 魔王S：混沌魔王（HP 300）

**战斗机制：**
- 玩家攻击力：基础20 + 段位加成
- 玩家血量：基础100 + 段位加成
- 答对题目：怪物扣除玩家攻击力的血量
- 答错题目：怪物回复10%血量并攻击玩家

### 4. 题目系统

**题目难度分级：**
- **D/D+级**：基础语法（变量、循环、条件语句）
- **C/C+级**：简单算法（排序、查找、字符串处理）
- **B/B+级**：中等算法（递归、动态规划入门）
- **A/A+级**：复杂算法（图论、高级DP）
- **S级**：高级算法（数学优化、复杂度优化）

**题目来源：**
1. 内置题库（50道精选题目）
2. LeetCode Easy/Medium题目
3. 自定义题目扩展

**难度递进机制：**
- 同一题目会根据段位调整难度
- D级显示完整提示和代码模板
- S级只显示题目要求，无提示
- **无时间限制**：玩家可以充分思考

**题目格式：**
```cpp
// 题目：求两数之和
// 输入：两个整数
// 输出：两数之和
// 限制：无时间限制，内存256MB
```

### 5. 多人游戏功能

**房间系统：**
- 创建房间：设置房间名称、段位限制、最大人数（2-10人）
- 加入房间：通过房间码或公开列表加入
- 游戏模式选择：个人竞技/团队模式

**实时功能：**
- 实时排行榜：显示当前房间内所有玩家进度
- 战斗动画：其他玩家答对时的攻击特效
- 聊天系统：简短的文字交流
- 观战模式：可以观看其他玩家战斗

**社交元素：**
- 好友系统：添加好友，查看好友战绩
- 成就系统：
  - 连胜成就
  - 解题速度成就
  - 特殊题目完成成就
- 分享功能：分享战绩到社交媒体

**奖励机制：**
- 每日登录奖励
- 连胜奖励
- 段位提升奖励

## 游戏流程

### 开始游戏流程
1. 玩家访问网站，输入昵称
2. 系统分配初始段位D
3. 选择游戏模式：
   - 单人模式：直接开始
   - 多人模式：创建/加入房间

### 单人游戏流程
1. 系统根据当前段位生成题目
2. 显示题目和怪物信息
3. 玩家编写代码并提交
4. 系统验证答案：
   - 正确：怪物扣血，获得经验
   - 错误：怪物回血，扣除玩家血量
5. 怪物血量归零：玩家获胜，提升段位
6. 玩家血量归零：游戏结束，记录失败

### 多人游戏流程
1. 房主创建房间，设置参数
2. 其他玩家加入房间
3. 所有玩家准备后开始游戏
4. 实时显示其他玩家进度
5. 游戏结束后显示排行榜
